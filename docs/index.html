Desktop-Browser erforderlich

Der Firmware-Installer funktioniert nur auf einem Desktop-Computer mit Google Chrome oder Microsoft Edge.

Bitte öffne diese Seite auf einem PC oder Mac.

TST Sensor Firmware-Update

Anleitung:

1. Klicken Sie auf „Verbinden".

2. Schließen Sie einen Sensor über das USB-Kabel an Ihren Computer an und prüfen Sie, welcher serielle Anschluss in der Liste angezeigt wird.

3. Wählen Sie diesen seriellen Anschluss aus und klicken Sie auf „Verbinden". Der Flash-Vorgang beginnt nun.

4. Warten Sie, bis die Meldung „Fertig" erscheint. Trennen Sie das Kabel während des Flash-Vorgangs nicht.

5. Sobald das erste Gerät fertig ist, trennen Sie es und wiederholen Sie den Vorgang mit dem nächsten Sensor.
